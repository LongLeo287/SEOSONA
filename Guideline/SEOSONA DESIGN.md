## 5. IMPLEMENTATION RULES

1. **CSS Variables in `:root`:** Always declare `--seosona-dark-1: #003566`, `--seosona-dark-2: #003366`, `--seosona-green: #46FF00`.
2. **Never pure `#000000`:** Use `#091338` (Charcoal) for dark text.
3. **Primary Button = Dark Navy**, not Neon Green. Neon Green is for **accents and glows only**.
4. **Gradient usage:** Reserved for backgrounds, progress indicators, and accent lines only.
5. **Poppins for all headings**, Inter for all body text.
6. **8px base grid**, spacing must be multiples of 8.
