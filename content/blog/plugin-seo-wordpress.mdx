## **Cách cài đặt các Plugin cho WordPress**

Để cài đặt các plugin mà tôi đã chia sẻ, bạn có thể chọn 2 cách: cài trực tiếp trên thư viện WordPress, cài đặt bằng file .zip Plugin

### **Cách 1: Cài plugin trực tiếp trên thư viện WordPress**

![các bước cài trực tiếp plugin seo wordpress](/images/blog/plugin-seo-wordpress-d575b9c2.jpg)

Các bước cài trực tiếp Plugin trên WordPress

*   Trong mục **Plugins** trên trang quản trị của WordPress, nhấn chọn **Add new**.

*   Gõ tên plugin cần cài đặt ở ô tìm kiếm, đợi plugin xuất hiện. Chọn **Install now** để cài đặt.

*   Đảm bảo bạn đã nhấn chọn **Activate** để thực hiện kích hoạt plugin.

### **Cách 2: Cài đặt file .zip Plugin có sẵn trong máy tính**

Cách cài đặt này thường dùng cho những plugin bạn cần trả phí và không có sẵn trong thư viện của WordPress, điển hình là WP Rocket.

![cài plugin seo wp bằng file zip](/images/blog/plugin-seo-wordpress-936c2e18.jpg)

Các bước cài Plugin bằng cách upload file zip

*   Vào **Plugins** => nhấn **Add New**
*   Nhấn chọn **Upload Plugin** và bắt đầu tải plugin từ máy tính.

*   Chọn **file .Zip** từ máy tính rồi click **Install Now**.
*   Chọn nút **Activate Plugin** để kích hoạt và hoàn tất cài đặt.
